# Lexical Analyzer

## Overview
This project implements a **Lexical Analyzer (Lexer)** for a simplified programming language called **PLUS**.  
The program reads a source file with the `.plus` extension, analyzes the tokens in the file, and produces an output file with the `.lx` extension containing the detected tokens.

## Key Features
- Recognizes predefined **keywords**
- Detects **identifiers**
- Supports **integer constants**
- Parses **string constants**
- Handles **operators and block symbols**
- Provides **basic lexical error detection**

## Technologies
- C Programming Language  
- GCC Compiler  
- Standard C Libraries
